1级标题

找样式，英文：style(css)

p标签有默认的style样式 display:block;翻译:显示，块

背景颜色样式:background-color

width： px翻译：宽度；作用：设置盒子的宽度

height： px翻译：高度；作用：设置盒子的高度

margin： px翻译：边距，也叫外边距；作用：设置盒子和其他盒子的距离

border： px翻译：边框；作用:设置盒子边框，默认透明，具体使用自行搜索

padding： px翻译：填充，也叫内边距；作用：设置盒子边框和内容之间的距离

标签的名称和作用用“html标签”搜索 推荐mdn 菜鸟教程

标签默认样式：display：block div=division翻译：分开、分隔

标签 alt：图片错误时显示的替代文字 src=source翻译：来源；资源

background 背景 color 颜色 #f1f1f1 颜色代码 text 文本 align 对齐

center 中间 padding 内边距 px pixel像素 font-size 字体大小

max 最大 width 宽度 margin 外边距 auto 自动 line 行 height 高度

css 负责管html标签的所有外观

text-align: center; html页面按F12 点击元素ph标签 在样式中使用可以居中

line-height:20px html页面行高代码 点击元素 在样式中直接使用
